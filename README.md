# Epoch

The relentless sands of time have never felt so heavy.

Epoch displays a clean visual to remind you not to procrastinate every time you open a new tab. It features an accurate age tracker and a clean, customizable design.

“Remember how long you have been putting off things, and how often you have received opportunities from the gods, and yet do not use it.”
― Marcus Aurelius, Meditations

## Features
- Watch your age tick forward in real-time, with up to 10 decimal places.
- Add an optional custom message or reminder to inspire your day.
- Native support for Dark and Light modes, plus full control over background, text, and label colors using custom Hex codes.
- Choose between fonts like JetBrains Mono, Outfit, Inter, and more.
